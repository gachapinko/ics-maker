# DESIGN.md — Monochrome Minimal UI

このファイルはデザインのトーン・ルールを定義する指示書です。
新しいページやコンポーネントを作成するときは、必ずこのガイドラインに従ってください。

---

## 🎨 カラーパレット

| 役割 | 値 | 用途 |
|---|---|---|
| Primary Black | `#111111` | テキスト・アイコン・アクティブ要素 |
| White | `#ffffff` | 背景・カード |
| Gray 100 | `#f5f5f5` | サブ背景・バナー背景 |
| Gray 200 | `#e8e8e8` | ボーダー・区切り線 |
| Gray 300 | `#d0d0d0` | 非アクティブボーダー |
| Gray 500 | `#888888` | サブテキスト・プレースホルダー |
| Gray 700 | `#444444` | セカンダリテキスト |

**アクセントカラーは原則使わない。** 使う場合は意味のある1色のみ（例：アクティブ状態を示す緑 `#22c55e`）。グラデーションは淡いパステル系のみ許容（雲・背景装飾など）。

---

## 🔤 タイポグラフィ

### フォント

| 役割 | フォント | 備考 |
|---|---|---|
| **ページタイトル** | `"ヒラギノ丸ゴ ProN", "Hiragino Maru Gothic ProN"` | システムフォント。`font-weight: 700` で使用 |
| **UI全般** | `'Noto Sans JP'` | 本文・ラベル・ボタンなど |

```css
/* ページタイトル */
font-family: "ヒラギノ丸ゴ ProN", "Hiragino Maru Gothic ProN", sans-serif;
font-weight: 700;

/* UI全般 */
font-family: 'Noto Sans JP', sans-serif;
```

```html
<!-- Google Fonts 読み込み（Noto Sans JP のみ） -->
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@300;400;500;700&display=swap" rel="stylesheet">
```

> ヒラギノはmacOS / iOSのシステムフォントのため追加読み込み不要。  
> Windows環境では自動的に `sans-serif` にフォールバックされる。

**ウェイト使い分け**
| ウェイト | フォント | 用途 |
|---|---|---|
| `700` | ヒラギノ角ゴシック | ページタイトルのみ |
| `700` | Noto Sans JP | セクション見出し・強調 |
| `500` | Noto Sans JP | カードタイトル・ボタンテキスト |
| `400` | Noto Sans JP | 本文・ラベル |
| `300` | Noto Sans JP | キャプション・補足テキスト |

### サイズ階層
| 用途 | サイズ | ウェイト |
|---|---|---|
| 大見出し（挨拶・名前） | 28–32px | Display体 |
| セクションタイトル | 18px | 600 |
| カードタイトル | 13–15px | 600 |
| 本文・ラベル | 12–13px | 400–500 |
| キャプション・補足 | 10–11px | 400 |

---

## 📐 スペーシング・レイアウト

- **画面の左右余白**: `20px`
- **カード内パディング**: `14–18px`
- **要素間の縦マージン**: `8px / 12px / 16px / 20px` の4段階で統一
- **コンポーネント間**: `12–16px`

余白は **詰めすぎず、広げすぎず**。情報の密度感を均等に保つ。

---

## 🃏 カードコンポーネント

```
背景色: #ffffff または #f5f5f5
ボーダー: 1.5px solid #e8e8e8  ← 影は使わない
角丸: 14–18px
パディング: 14–18px
```

- **シャドウは基本使わない**。深度はボーダーと背景色の差で表現する。
- カードのネストは最大2段まで。

---

## 🔘 ボタン・チップ

### アクティブ（選択中）
```
背景: #111111
テキスト: #ffffff
ボーダー: none
角丸: 20px（pill型）
```

### 非アクティブ
```
背景: #ffffff
テキスト: #111111
ボーダー: 1.5px solid #d0d0d0
角丸: 20px
```

### ムードボタン・グリッドボタン
```
背景: #ffffff
ボーダー: 1.5px solid #e8e8e8
角丸: 12px
パディング: 10px 14px
ホバー: 背景 #f5f5f5
```

---

## 🧭 ナビゲーション

- ボトムナビは**アイコンのみ** or **アイコン＋小さいラベル**
- アクティブ状態は色ではなく**サイズ・ウェイトの変化**で示す
- ボーダーは `1px solid #e8e8e8`（シャドウなし）

---

## 🖼 画像・メディア

- 写真はカード内に収め、**角丸 16px**
- 白黒または低彩度の写真が馴染む
- 動画プレビューは**ダーク背景（#111111）**上に白テキスト・アイコンで表示
- プロフィール写真の「いいね」ボタンは右下に `rgba(255,255,255,0.9)` の丸ボタンとして重ねる

---

## ✨ アイコン

- **線画スタイル（stroke）** を使う。塗りつぶしアイコンは避ける。
- ストローク幅: `1.5px`
- サイズ: `16–22px`（コンテキストによる）
- ライブラリ推奨: `lucide-react` / `phosphor-icons`

---

## 🌥 装飾・雰囲気

- 装飾的な背景（雲、グラデーション）は**薄いパステル**のみ許容
  ```css
  background: linear-gradient(160deg, #f0eefc 0%, #fdf4ee 100%);
  ```
- ノイズテクスチャ・強いシャドウ・蛍光色は**使わない**
- アニメーションは `fadeIn + translateY` の控えめなものだけ

---

- [ ] シャドウを使っていないか（使う場合はシェルの外側のみ）
- [ ] ページタイトルはヒラギノ丸ゴ ProN `font-weight: 700` か
- [ ] UI全般は Noto Sans JP か（ウェイト 300/400/500/700 で階層表現）
- [ ] ボーダーは `1.5px solid #e8e8e8 or #d0d0d0` か
- [ ] アイコンは線画スタイルか
- [ ] ボタンのアクティブ状態は黒背景・白テキストか
- [ ] 余白は 8/12/16/20px の階段で統一されているか
